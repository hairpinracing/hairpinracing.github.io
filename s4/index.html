01

How It Works

Each team has 2 drivers. Each race runs one class pairing — either Gr1–Gr2 or Gr3–Gr4. Both teammates race every week, one in each class within that pairing: in a Gr1–Gr2 race, one drives Gr1 and the other drives Gr2. Teams decide internally who takes which class.

"Teams change driver class" markers in the schedule indicate the points where teammates swap roles within the pairing, so each driver covers all four classes by end of R4 and again by end of R9.

Across the season, each driver will race every class (Gr1, Gr2, Gr3, Gr4) twice. R1–R4 is one full rotation. R5 is the first "all-classes" race, assigned by standings. R6–R9 is another rotation (your team flipped to the opposite class from R5). R10 is the second "all-classes" race, opposite of R5.

01

How It Works

Each team has 2 drivers. In every race, one teammate runs Gr1–Gr2 and the other runs Gr3–Gr4. Teams decide internally who drives which class each race.

"Teams change driver class" markers in the schedule indicate the points where driver pairings within a team swap so each person covers all four groups.

03

R5 — Class Assignment

Based on season standings after R4. Alternating down the standings:

P1→Gr1–Gr2
P2→Gr3–Gr4
P3→Gr1–Gr2
P4→Gr3–Gr4
P5→Gr1–Gr2
P6→Gr3–Gr4
P7→Gr1–Gr2
P8→Gr3–Gr4

Your team picks internally which teammate drives which class.

After R5, your team flips class for R6–R9. If you were Gr1–Gr2 at Spa, you're Gr3–Gr4 the rest of the way. Each driver still gets one race in each class across R6–R9.

R10 works like R5, but your team is in the opposite class from R5.
